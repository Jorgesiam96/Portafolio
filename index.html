Projetos em destaque

  • BurgerDev "Burger DevClub é um projeto web que se destaca pelo seu meticuloso foco na implementação de tecnologias front-end. Desenvolvido inteiramente com HTML, CSS e JavaScript, este projeto exibe um menu interativo que mostra uma variedade de hambúrgueres e pratos com um design visualmente atraente e funcionalidades intuitivas. O código HTML é usado para estruturar o conteúdo do menu, enquanto o CSS é responsável por estilizar cada elemento, desde o layout até os detalhes de design. Por fim, o JavaScript é empregado para adicionar interatividade, permitindo que os usuários explorem o menu de forma dinâmica e façam pedidos com facilidade."

  • JokenpoDev "Jokenpo é um projeto de jogo simples que traz a clássica diversão do pedra, papel e tesoura para o mundo digital. Desenvolvido usando HTML, CSS e JavaScript, este jogo oferece uma experiência interativa onde os jogadores podem competir contra o computador em partidas emocionantes. O HTML é utilizado para estruturar a interface do jogo, enquanto o CSS é responsável por estilizar os elementos e dar vida ao design, o JavaScript é empregado para implementar a lógica do jogo, permitindo que os jogadores escolham sua jogada e determinando o vencedor de cada rodada."

  • InstaDev "Meu primeiro projeto é uma página de login inspirada no Instagram, onde explorei os conceitos fundamentais de HTML, CSS e JavaScript para criar uma experiência de usuário semelhante à da rede social popular. Utilizando HTML para estruturar os elementos da página, CSS para estilizar o layout e JavaScript para adicionar funcionalidades interativas, como validação de formulário e animações de transição, consegui replicar a essência da página de login do Instagram."

  • ShoopingDev "ShoppingDev marca o início da minha jornada no curso DevClub, onde, junto com meu mentor Rodolfo Mori, comecei a explorar o emocionante mundo do desenvolvimento web.
